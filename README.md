# 3D Perspective Drawing Tool (Python)

基於Python的3D立方體互動工具，用於模仿並改進練習繪畫時的空間感訓練工具。

## 功能:
- **滑鼠操控**：
  - **旋轉控制 (滑鼠左鍵)**：具備「單軸鎖定」機制，會根據初始滑動方向自動鎖定在水平或垂直旋轉。
  - **位移控制 (滑鼠右鍵)**：可平移立方體位置，同樣支援單軸鎖定。
  - **快速歸位 (Space)**：一鍵將立方體重置回畫面中央與初始角度。
  - **方塊軌跡 (滾輪)** : 可以目前的透視畫面移動方塊的的軌跡
  - **縮放功能 (Ctrl + 滾輪)**：可縮放立方體比例。 

- **快捷鍵操作**：  
  - **透明模式 (T)**：切換立方體透明度。
  - **消失點輔助線 (G)**：沿著立方體邊緣延伸出無限長的直線，顯示收束於消失點的線。
  - **座標軸顯示 (A)**：顯示 X (紅)、Y (綠)、Z (藍) 三色世界座標軸
  - **動態視平線 (H)**：顯示消失點位置的第地平線

## 檔案架構
- **main.py**：負責視窗管理、OpenGL 初始化、視窗縮放處理、事件監聽及主渲染循環。
- **models.py**:定義 Cube 類別，包含頂點數據、顏色字典、實心面、強化結構線與消失點輔助線的繪製邏輯。
- **/assets**:存放UI圖片的資料夾

## 模組化標籤架構
- **[SETUP]**：字體與環境效能設定。
- **[PROJECTION]**：處理 3D 透視投影與視窗比例。
- **[SCENE]**：負責動態視平線與基準線渲染。
- **[PERSPECTIVE]**：消失線無限延伸運算邏輯。
- **[COLLISION]**：3D 頂點轉 2D 螢幕點擊判定。
