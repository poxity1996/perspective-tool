# 3D Perspective Drawing Tool (Python)

基於Python的3D立方體互動工具，用於模仿並改進練習繪畫時的空間感訓練工具。

## 功能:
- **旋轉控制 (左鍵)**：拖曳可旋轉立方體。具備「單軸鎖定」機制，會根據初始滑動方向鎖定在水平或垂直旋轉，避免旋轉過度混亂。
- **位移控制 (右鍵)**：拖曳可移動立方體位置，同樣具備「水平/垂直單軸鎖定」。
- **快速歸位 (空白鍵)**：按下空白鍵即可將立方體重置回畫面中央與初始角度。
- **座標軸顯示**：內建紅(X)、綠(Y)、藍(Z)三色座標軸，幫助理解空間指向。
